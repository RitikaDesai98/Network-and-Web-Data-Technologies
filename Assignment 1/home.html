Welcome!

This website is desgined using HTML, CSS and Javascript.

This website contains 4 pages with a variety of applications.
  1. Tasks Page:
    1. A list of tasks can be seen that are added by default to the code
    2. In order to add a new task, enter the task name in the input box and click on the add button
    3. To mark a task as complete or incomplete, simply toggle on the checkbox
  2. Budget Page:
    1. An entry in the budget table has already been made
    2. In order to add a new expense, enter the expense name in the first input box and the amount spent in the second input box and click on the add button
    3. To get the total amount spent, click on the 'Calculate Total' button
  3. Recipe Page:
    1. A recipe card has already been made
    2. In order to add a new recipe, enter the ingredients in the first input box and the recipe in the second input box and click on the save button
  4. Calculator Page:
    1. A calculator has been created with the basic operations like add, subtract, multiply, divide. Decimal numbers are possible
    2. To evaluate an expression, simply input the number followed by the operator followed by another number and click on the = sign
    3. To clear the screen, click on the clear all button

Hope this website is of some use to you and helps you in ways you're seeking.
